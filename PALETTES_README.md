# Пользовательские палитры цветов

## Как использовать

1. **Открыть менеджер палитр**: Нажмите кнопку "Palette" на панели инструментов
2. **Создать новую палитру**: 
   - В менеджере палитр нажмите "Новая"
   - Введите имя палитры
   - Нажмите OK
3. **Добавить цвет в палитру**:
   - Выберите палитру в комбо-боксе
   - Нажмите "Добавить цвет"
   - Выберите цвет в диалоге выбора цвета
4. **Удалить цвет из палитры**:
   - Выберите палитру и нажмите на цвет
   - Нажмите "Удалить цвет"
5. **Удалить палитру**:
   - Выберите палитру и нажмите "Удалить"
   - Нельзя удалить последнюю палитру
6. **Сохранить изменения**: Нажмите "Сохранить" в менеджере

## Где хранятся палитры

Палитры сохраняются в папке `Palettes` в директории приложения в формате JSON.

Пример структуры файла палитры:
```json
{
  "Name": "My Palette",
  "Colors": [
    -16777216,
    -1,
    -8355712
  ]
}
```

Каждый цвет хранится как 32-битное целое число в формате ARGB.
